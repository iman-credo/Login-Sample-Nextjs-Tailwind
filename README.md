# Login-Page-Template
A modular login page sample made by Next.js , using Tailwindcss as CSS framework.

## Usage
Clone the repository on your local machine.
run command:
```
npm install 
```
or
```
yarn install
```
so that node_modules folder appears in the project directory.
then run command:
```
npm run dev
```
